
Dino Math Adventure  
A kid friendly math learning app built with .NET MAUI
Dino Math Adventure is an educational mobile app designed to help young learners practice early math skills in a fun, interactive, dinosaur themed environment. The app focuses on simple, confidence building exercises like counting, addition, and subtraction. The learning experience is wrapped in bright visuals and playful interactions that keep kids engaged.
This project represents the **public, portfolio safe version** of the app, with all proprietary assets removed and the core logic preserved for demonstration.
 Features
- Kid friendly UI with large buttons and simple navigation  
- Counting practice with visual prompts  
- Addition and subtraction exercises  
- Positive reinforcement to encourage learning  
- Touch friendly interactions designed for small hands  
- Cross platform support (Android, iOS, Windows) via .NET MAUI  
Tech Stack:
Technology	Purpose
.NET MAUI	Cross platform UI framework
C#	Application logic
XAML	UI layout and styling
MVVM pattern	Clean separation of UI and logic

 Running the Project

1. Install the latest .NET SDK (with MAUI workload)  
2. Clone the repository:

git clone https://github.com/smittydumbell22/DinoMathAdventure-Public.git

3. Open the project in Visual Studio 2022 or VS Code  
4. Select your target platform (Android, Windows, etc.)
